# Submission guidelines
1. When describing your project, please provide a concise and objective description of what it does. Avoid using promotional language such as "best LCU library". Make sure your description is grammatically correct and free of spelling errors.
2. Ensure that your project is relevant to League of Legends development, tools, or community. Project should be dedicated to League of Legends or its ecosystem.
3. Make sure the submitted project aligns by Riot Games' stance on third-party tools. **Projects that are obvious exploits and/or violate game's integrity will not be accepted.** Such cases include but are not limited to:
   - Automated actions in-game
   - Getting access to game data you should not have access to (like lobby name revealing, if something is purposefully hidden from player, it's usually for a reason)
   - Manipulating game in a way that gives unfair advantage over other players
   - Ruining the experience of other players: chat exploits, spammers, crashers, etc.
4. Please provide a link to the project's repository (e.g., GitHub, GitLab) or official website.


# How to contribute
If you have something awesome to contribute to an awesome list, this is how you do it.

You'll need a [GitHub account](https://github.com/join)!

1. Access this list's GitHub page.
2. Click on the `readme.md` file: ![Step 2 Click on Readme.md](https://cloud.githubusercontent.com/assets/170270/9402920/53a7e3ea-480c-11e5-9d81-aecf64be55eb.png)
3. Now click on the edit icon. ![Step 3 - Click on Edit](https://cloud.githubusercontent.com/assets/170270/9402927/6506af22-480c-11e5-8c18-7ea823530099.png)
4. You can start editing the text of the file in the in-browser editor. Make sure you follow guidelines above. You can use [GitHub Flavored Markdown](https://help.github.com/articles/github-flavored-markdown/). ![Step 4 - Edit the file](https://cloud.githubusercontent.com/assets/170270/9402932/7301c3a0-480c-11e5-81f5-7e343b71674f.png)
5. Say why you're proposing the changes, and then click on "Propose file change". ![Step 5 - Propose Changes](https://cloud.githubusercontent.com/assets/170270/9402937/7dd0652a-480c-11e5-9138-bd14244593d5.png)
6. Submit the [pull request](https://help.github.com/articles/using-pull-requests/)!
